# A/B Testing Statistical Analysis
Python project analyzing A/B test results to compare user behavior between control and test groups.

## Business Task
Analyze experiment data to understand how user behavior changes between two groups and check if the differences are statistically significant.

## Tools
Python, Pandas, SQL, Google BigQuery, Statsmodels, A/B Testing

## Key Insights
- User behavior differs between the control and test groups.
- Some conversion metrics improved in the test group.
- Statistical tests help determine if the changes are significant.
- Experiment analysis helps evaluate product changes.
